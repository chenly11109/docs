# 埋点
## 定义
所谓埋点就是在应用中特定的流程收集一些信息，用来跟踪应用使用的状况，后续用来进一步优化产品或是提供运营的数据支撑，  

### 包括访问（Visits)，   
访客（Visitor）， - session,   
停留时间（Time On Site），  
页面查看（Page Views，又称为页面浏览)  
跳出率（Bounce Rate，又可称为蹦失率） - 单个页面访问（只打开一个页面就关掉了）所占的会话比例

推出率，转化率，

### 信息收集可以大致分为两种:
页面统计（track this virtual page view），
统计操作行为（track this button by an event）。
